# Builder Dashboard

Version : 1.0

Écran : BD-001

Priorité : Critique

---

# Pourquoi cet écran existe

Le Dashboard est l'écran que le conseiller verra le plus souvent.

Il doit donner immédiatement l'impression :

"Je maîtrise tous mes dossiers."

Il ne doit jamais ressembler à un CRM.

Il doit ressembler à un cockpit.

---

# Objectif

Permettre au conseiller de retrouver un dossier en moins de 5 secondes.

Créer un nouveau dossier en un clic.

Visualiser immédiatement les dossiers prioritaires.

# Objectif psychologique

Le conseiller doit avoir envie d'ouvrir le logiciel tous les matins.

Le Dashboard doit procurer un sentiment :

de maîtrise

de rapidité

de professionnalisme.

Jamais de surcharge.

Jamais plus de trois niveaux d'information.

---

# Critères d'acceptation

✓ Création dossier en un clic

✓ Recherche instantanée

✓ Ouverture dossier < 1 seconde

✓ Responsive

✓ Navigation clavier

✓ Accessible
