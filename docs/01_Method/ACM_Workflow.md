# ACM Studio

# ACM Interactive Workflow

Version : 1.0

Statut : Fondation

Priorité : Critique

---

# Objectif

Décrire le déroulement complet d'une Analyse Comparative de Marché depuis la création du dossier jusqu'à la signature du mandat.

Ce document constitue le workflow métier de référence.

Aucun développement ne peut s'écarter de ce document sans modification de version.

---

# Philosophie

Le conseiller ne vend jamais un prix.

Le conseiller accompagne une prise de conscience.

Chaque étape possède un objectif précis.

Chaque écran possède un objectif précis.

Chaque interaction possède un objectif précis.

# PHASE 1

Premier rendez-vous

Objectif

Comprendre le projet vendeur.

Aucune estimation définitive.

Aucune présentation.

Le conseiller collecte uniquement les informations.

---

• Photos du bien

• Description

• Adresse

• Surface

• Nombre de pièces

• Prestations

• Particularités

• Historique

• Motivation

• Projet vendeur

# LE BIEN VENDEUR

Toujours présenté en premier.

Jamais de prix.

Jamais d'estimation.

Objectif psychologique

"Oui, c'est bien mon bien."

# LES CONCURRENTS

Toujours un seul concurrent à la fois.

Jamais deux concurrents sur une même slide.

---

Ordre

Concurrent 1

↓

Concurrent 2

↓

Concurrent 3

↓

...

---

Pour chaque concurrent

Slide A

Présentation.

Photo.

Caractéristiques.

Sans prix.

Sans délai.

Question :

"Ce bien vous semble-t-il comparable au vôtre ?"

Le conseiller recueille la réponse.

---

Slide B

Même slide.

Nouvelle question.

"À votre avis, quel est son prix ?"

Le vendeur répond.

Le conseiller saisit le prix estimé.

---

Slide C

Même slide.

Le prix réel apparaît.

Question :

"Ce prix vous paraît-il cohérent ?"

Discussion.

---

Slide D

Même slide.

Le délai.

La baisse de prix.

Les modifications.

Question

"Pourquoi ce bien est-il encore disponible ?"

Le vendeur répond.

Le conseiller écoute.

Jamais de débat.

---

Fin du concurrent.

On passe au suivant.

---

# APRÈS LES CONCURRENTS

Le vendeur connaît désormais le marché.

On ne parle toujours pas de son bien.

---

# QUESTION 1

Parmi tous ces biens,

lequel vous semble être le plus dangereux ?

Le conseiller sélectionne.

---

# QUESTION 2

Pourquoi ?

Prix

Secteur

# QUESTION 3

Selon vous,

quel serait aujourd'hui le bon prix de mise en vente de votre bien ?

Le conseiller saisit le montant.

Aucun commentaire.

# ESTIMATION

Uniquement maintenant.

# MANDAT

Le conseiller répond aux dernières questions.

Présente son accompagnement.

Propose le mandat.

---

# APRÈS LE RENDEZ-VOUS

Le dossier reste disponible.

Toutes les réponses du vendeur sont enregistrées.

Elles alimentent :

Le rapport conseiller.

Les statistiques.

L'IPC.

L'historique.

---

# RÈGLES ABSOLUES

Jamais d'estimation avant la lecture du marché.

Jamais de prix du bien vendeur avant la fin.

Jamais plusieurs concurrents sur une même slide.

Toujours demander avant de révéler.

Toujours laisser le vendeur répondre.

Toujours privilégier les données réelles.

Toujours conclure par l'expertise du conseiller.

L'IA prépare.

Le conseiller décide.

Le vendeur comprend.
