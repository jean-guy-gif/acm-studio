# Pourquoi ce document existe

Le Storyboard définit le déroulement exact du rendez-vous vendeur.

Chaque slide possède :

- un objectif psychologique
- un objectif métier
- un objectif graphique
- un objectif commercial

Une slide = une idée.

Une transition = une progression.

Le vendeur ne doit jamais avoir l'impression de regarder une présentation.

Il doit avoir l'impression de découvrir progressivement la réalité de son marché.

# Les règles absolues

Jamais deux concurrents sur une même slide.

Jamais révéler un prix avant la question.

Jamais annoncer l'estimation avant l'Acte 6.

Toujours laisser le vendeur répondre.

Toujours afficher les vraies photos.

Toujours conserver le même design.

Toujours terminer par une décision.

---

# Ce que mémorise l'IA

Chaque réponse.

Chaque prix estimé.

Chaque concurrent préféré.

Chaque justification.

Chaque hésitation.

Chaque commentaire.

Ces données alimentent le Baromètre de perception.

---

# Objectif final

À la fin du rendez-vous, le vendeur doit avoir le sentiment que :

- la conclusion lui appartient ;
- le marché a guidé sa réflexion ;
- l'estimation est logique ;
- le conseiller est un expert qui s'appuie sur des faits.

Le logiciel disparaît derrière la qualité de l'échange.
